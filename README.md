# Nikhil ❤️ Tanisha — Digital Wedding Invitation

एक सुंदर और प्रीमियम डिजिटल वेडिंग इनविटेशन वेबसाइट, जिसे 'Royal Gold & Maroon' थीम पर आधारित बनाया गया है।

## ✨ फीचर्स
* **Responsive Design:** यह मोबाइल और डेस्कटॉप दोनों पर सही से चलती है।
* **Animations:** इसमें गिरते हुए फूलों की पंखुड़ियाँ (Petals) और चमकते हुए कण (Particles) शामिल हैं।
* **Scroll Reveal:** स्क्रॉल करने पर कंटेंट स्मूथ तरीके से दिखाई देता है।
* **Designer Popup:** इसमें 'Pixel_invitess' के लिए एक इंटरैक्टिव कॉन्टैक्ट पॉपअप दिया गया है।

## 🛠️ कैसे इस्तेमाल करें
1. इस रिपॉजिटरी को क्लोन करें।
2. `index.html` को किसी भी ब्राउज़र में खोलें।
3. गिटहब पेजेस (GitHub Pages) के जरिए इसे लाइव होस्ट करें।
